# Security Policy

This repository is published as a public, read-only reference implementation.

## Supported Versions

No release line is actively supported.

## Reporting a Vulnerability

- Do not assume this repository is actively monitored for vulnerability reports.
- GitHub Issues and pull requests may be disabled as part of the read-only setup.
- If you need a maintained variant, fork the repository and operate your own review and release process.

If a private disclosure path is needed in the future, document it here before enabling public intake.
